# Retro VHS Snake Game

A classic Snake game reimagined with a retro low-poly aesthetic and VHS visual effects.

## Controls

| Action | Key(s) |
| :--- | :--- |
| **Move Up** | `W` or `Arrow Up` |
| **Move Down** | `S` or `Arrow Down` |
| **Move Left** | `A` or `Arrow Left` |
| **Move Right** | `D` or `Arrow Right` |
| **Pause Game** | `ESC` |
| **Select / Resume** | `ENTER` |
| **Navigate Menus** | `W` / `S` or `Arrow Up` / `Arrow Down` |

## Features

- **Classic Gameplay**: Eat the apple, grow your snake, and avoid hitting walls or yourself.
- **Retro Aesthetic**: CRT scanlines, screen curvature, and chromatic aberration effects.
- **Low Poly Visuals**: Pseudo-3D cube rendering for the snake and food.
- **8-Bit Audio**: Synthesized background music and sound effects (eat, crash, menu).
- **Menus**: Fully navigable Start, Pause, and Game Over menus.

## How to Run

Simply open `index.html` in any modern web browser.
